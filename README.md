## StartPage for CloudPhone

A lightweight web-search browser widget with a search bar, favorites, history and settings, built for CloudPhone feature phones (240x320 / 128x160 screens) with LSK (Menu) / RSK (Back) softkey navigation.

## Add to CloudPhone
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).
2. Select **Add Widget**.
3. Paste this Start URL:
   `https://tasmon.github.io/StartPage`
4. Upload icons if required.
5. Save and refresh — **StartPage** will appear on your CloudPhone.
